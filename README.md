# Tasks Repository Overview

This repository contains all the module tasks for the course, organized systematically by module and task type. The purpose of this repository is to provide a clear and accessible collection of all individual and group assignments for reference, review, and submission purposes.

Each module contains two types of tasks:  
- *Individual Tasks:* Completed individually by the student, focusing on personal analysis, exploration, or data collection.  
- *Group Tasks:* Completed collaboratively, highlighting teamwork and combined understanding of complex concepts.

## Repository Structure

### Module-1
- *Individual-Tasks:* Compare different forms of intelligence (human, animal, machine) using a chart or diagram.  
- *Group-Tasks:* AI – a threat or a boon to humanity?

### Module-2
- *Individual-Tasks:* My Daily Data Inventory – tracking and classifying daily data interactions.  
- *Group-Tasks:* Big Data Processing Map – mapping the flow of data in a real-world system.

### Module-3
- *Individual-Tasks:* Exploration – practical experiments or analyses to understand patterns or relationships.  
- *Group-Tasks:* A Simple ML Process Flow – creating a complete machine learning workflow as a team.

---

All tasks are provided as *PDF files* in their respective folders. This structure ensures easy navigation, quick access to each assignment, and clear documentation of all work completed throughout the course.
